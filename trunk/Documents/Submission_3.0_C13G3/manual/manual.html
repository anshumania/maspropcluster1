Each Hidato puzzle starts with a grid partially filled with numbers. The goal is to fill the grid with consecutive numbers that connect horizontally, vertically or diagonally.

Hidato tips:

  • •.The first and last numbers a puzzle are circled. 

  • •.It is not necessary to start from thr first number. Sometimes it is better to start elsewhere. 

  • •.Working backward (counting down in numbers) can reveal key clues to solving the puzzle. 

1.2 Running application

This Hidato Application is provided under the Apache 2 Open Source license. It can be freely distributed with due acknowledgement to the developers.

The Application is provided as standalone executable jar and requires the java runtime environment of 1.6 and above to run.

It can be run by simply double clicking on the executable jar.

  1. 1.Double click on the Hidato.jar 

  2. 2.Click the File menu and the click the New game. 

  3. 3.Options for game generation will be shown. You can choose board size (x-dimension, y-dimension), difficulty level (easy, medium, hard) and board shape (rectangular, oval, donut). 

  4. 4.Press Continue to start a new game. 

  5. 5.The timer will be begin immediately. 

  6. 6.Select a cell and type in a number. Continue to fill up the grid. 

  7. 7.When you have finished, click Done. 

2.2.1 Entering/deleting values to/from cell

  • •.To enter value select cell by clicking on it. Enter number from 1 to N using keyboard (N – number of active cells) 

  • •.To delete value select cell by clicking on it. Remove number using keyboard 

3 Saving and loading games

  1. 1.If you want to save game, click on Save game in File menu 

  2. 2.Enter name for the file to save your Hidato game into and select the directory to save it into 

  3. 3.Click Save 

  4. 4.For loading game, click Load game in File menu 

  5. 5.Navigate to the directory where you have saved your previous game and select the file 

  6. 6.Click Open 

To view high score click on Scores in File menu.
