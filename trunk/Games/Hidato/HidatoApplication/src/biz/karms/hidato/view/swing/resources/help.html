Hidato: How to Play

Hidato is a puzzle game invented by Dr. Gyora Benedek, an Israeli mathematician. The goal of Hidato is to fill the grid with consecutive numbers that connect horizontally, vertically, or diagonally.

To Play Hidato

  1. Double-click on the Hidato jar.
  2. Click the File menu, and the click New game.
  3. Options for game generation will be shown. You can choose board size (x-dimension, y-dimension), difficulty level (easy, medium, hard) and board shape (rectangular, oval, donut).
  4. Press Continue to start a new game.
  5. The timer will be begin immediately.
  6. Select a cell and type in a number. Continue to fill up the grid.
  7. When you have finished, click Done

Entering/deleting values to/from cell

  • To enter value select cell by clicking on it. Enter number from 1 to N using keyboard (N - number of active cells).
  • To delete value select cell by clicking on it. Remove number using keyboard.

Saving and loading games

  1. If you want to save game, click on Save game in File menu.
  2. Enter name for the file to save your Hidato game into and select the directory to save it into.
  3. Click Save.
  4. For loading game, click Load game in File menu.
  5. Navigate to the directory where you have saved your previous game and select the file.
  6. Click Open.

View high scores

To view high score click on Scores in File menu.
