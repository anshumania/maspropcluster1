Sudoku: How to Play

Sudoku is a combinatorial placement puzzle and is solved using deductive logic. The objective of Sudoku is to fill a N�N grid so that each column, each row, and each of the N, ( with size squareRootN�squareRootN ), zones contains the digits from 1 to N only once. A valid Sudoku has only one unique solution.

To Play Sudoku

1. Double-click on the Sudoku jar.
- If the Sudoku jar is not available, you might need to install the game. For more information about installing the Sudoku jar, see the Installation Guide provided
- By default you are shown a blank Sudoku empty grid.

2. Click the File Menu, and the click New game

- You will be a shown options for choosing your Sudoku Variant and Difficulty Level. By default the Sudoku Variant is set to 9x9 and the Difficulty level is set to Extreme.

3. Press Ok to start a new game

4. The timer will be begin immediately

5. Select a square and type in a number. Continue to fill up the grid.

6. When you have completed the Sudoku Game click done


To enter and delete values from a Cell

1. Select a Cell by clicking on it
2. Enter a number from 1 to N ( for a NxN puzzle ).
3. Press Enter
4. To remove a number from a Cell, select the cell by clicking on it
5. You can use the delete, backspace and the spacebar to remove the value from the cell


Tips for GamePlay

1. If you wish to turn on hints, click on the Sudoku Menu and click on Hints to check the box against it.
- If you wish to turn off hints simply click on Hints again to uncheck the box next to it.
2. To solve a Sudoku game click on the Sudoku Menu and click on Solve.

To Save Sudoku games and continue saved games

1. If you need to finish a game later just click on the File Menu and click Save game
2. Enter a name for the file to save your Sudoku game into and select the directory to save it into
3. Click Save
4. To continue with a saved game, click on the File Menu and click Load game
5. Navigate to the directory where you have saved your previous game and select the file.
6. Click Open

To create your own Sudoku

1. If you wish to create your own Sudoku click on the Sudoku Menu and click Create game
2. Choose the variant of the Sudoku which you wish to create. The options will be
- 4x4, a Sudoku with 2x2 zones
- 9x9, a Sudoku with 3x3 zones
- 16x16, a Sudoku with 4x4 zones
3. Click on CreationSettings Tab.
4. Choose the percentage of the Sudoku grid that you would want to fill.
- You can leave the grid blank by choosing 0 percent and fill upto 70 percent.
- The numbers filled in would be placed in all valid positions according to the rules of Sudoku.
5. Click Ok
- You will be presented with a grid with a few given numbers according to your Creation settings.
6. Now you can start filling in numbers into the Grid.

Tips for Creation

1. Click on Sudoku Menu and Click on Validate .
- This will tell you if your partially filled Sudoku grid is a valid Sudoku or not, that is it has one solution or not.
